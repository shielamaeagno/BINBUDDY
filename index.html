ℹ️ About

About BinBuddy

🌿 What is BinBuddy?

BinBuddy is a smart waste tracking and rewards platform designed to help households in Lipa City, Philippines properly segregate their waste and earn EcoPoints for doing so. It connects households, waste collectors, and barangay administrators in one seamless platform.

⚖️ Republic Act No. 9003

The Ecological Solid Waste Management Act of 2000 mandates the proper collection, transport, and disposal of solid waste. BinBuddy helps every Filipino household comply with RA 9003 by making waste segregation easy, rewarding, and community-driven.

RA 9003 DENR Compliant QC-Aligned

🌍 SDG 12 – Responsible Consumption & Production

BinBuddy directly supports UN Sustainable Development Goal 12 by promoting responsible waste disposal habits, reducing landfill burden, and increasing the recycling rate in Metro Manila from its current 30% toward the 2030 target of 80%.

📊 Philippine Waste Context

12M Households in Metro Manila 9,500 tons waste/day Only 30% recycled Lipa City: 3.1M population 142 Barangays in QC

💼 Business Model

Barangay Subscription Plans: Free, Basic (₱1,500/mo), Premium (₱5,000/mo) with advanced analytics and priority support.

Sponsorship & Advertising: Local businesses can advertise on the platform to reach eco-conscious households.

Recycling Commission Analytics: Detailed reports for junk shops and recycling partners to optimize operations.

📄 Data Structure

XML / XSLT Viewer

📄 XML Data – BinBuddy uses XML to store waste logs, users, and rewards. The XSLT file transforms this data into readable HTML reports.
📄 Sample: Waste Log XML
<?xml version="1.0" encoding="UTF-8"?>
<binbuddy>
  <wasteLogs>
    <log id="LOG001" userId="USR001">
      <type>biodegradable</type>
      <category>Food Scraps</category>
      <quantity unit="kg">1.2</quantity>
      <date>2026-04-14</date>
      <method>manual</method>
      <verified>true</verified>
      <ecoPointsEarned>25</ecoPointsEarned>
    </log>
  </wasteLogs>
</binbuddy>
🔄 XSLT Transformation
<xsl:stylesheet version="1.0">
  <xsl:template match="log">
    <tr>
      <td><xsl:value-of select="@id"/></td>
      <td><xsl:value-of select="type"/></td>
      <td><xsl:value-of select="quantity"/></td>
      <td><xsl:value-of select="ecoPointsEarned"/></td>
    </tr>
  </xsl:template>
</xsl:stylesheet>
✅ The full XSLT file (/xsl/binbuddy-transform.xsl) transforms all XML data into an HTML report table showing users, logs, rewards, and barangay stats.
🗄️ SQL Database Schema
CREATE TABLE waste_logs (
  log_id      VARCHAR(10) PRIMARY KEY,
  user_id     VARCHAR(10) NOT NULL,
  waste_type  ENUM('biodegradable','recyclable','residual','special'),
  quantity    DECIMAL(6,2),
  log_date    DATE,
  is_verified BOOLEAN DEFAULT FALSE,
  eco_points  INT DEFAULT 0,
  FOREIGN KEY (user_id) REFERENCES users(user_id)
);

SELECT waste_type, SUM(quantity)
FROM waste_logs
WHERE user_id = 'USR001'
GROUP BY waste_type;
